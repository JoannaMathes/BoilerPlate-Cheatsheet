HTML Boiler Plate Cheatsheet

Understand the essentials of building an html webpage and how each element works.

HTML Tag What it does Where & how to use it
<!DOCTYPE html> This tag informs the browser what language to expect Placed at the top of every html webpage. Note that there is no closing tag. All other code goes underneath it
<html> </html> This tag holds all the html code on the page Use it below the <!DOCTYPE html> tag and put all other code inside of it
<head> </head> This tag holds all the metadata which is information that isn't visible on the page but is needed behind the scenes to build the page (such as stylesheets and scripts etc). Place this tag below the opening <html> and above the <body> tag.
<meta charset="UTF-8"> This tag is a meta data tag and defines the character encoding for the html document, this defines how the computer interprets each character and symbol. UTF-8 is the general standard. Place this tag inside the <head> tag. And note that this tag is self closing.
<meta name="viewport" content="width=device-width, initial-scale=1.0"> This tag is a meta data tag and sets the width of the page equal to the width of the device screen that it is being displayed on. Place this tag inside the <head> tag. And note that this tag is self closing.
<title> </title> This tag sets the title for the page which is displayed in the browser tab. Place this tag inside the <head> tag.
<link href="#" rel="#" type="#"> This tag allows you to link stylesheets, fonts, scripts and more. Place this tag inside the <head> tag.
<body> </body> This tag is where all the visible content is written. Place this tag below the <head> tag and the closing tag should be above the closing </html>
